# Bank Transaction Fraud Detection Dashboard 🏦 (Group Project)

## Project Overview

This project is an interactive Power BI dashboard developed as a group project using Agile methodology to analyze bank transaction fraud patterns and identify important business insights.

The dashboard transforms raw transaction data into meaningful visual insights and helps understand fraud behavior through customer demographics, transaction patterns, geographical analysis, and time-based trends.

The purpose of this project is to support fraud monitoring and assist in identifying suspicious activities through data-driven analysis.

## Problem Statement

Financial institutions process a large number of transactions every day, making fraud detection difficult.

The objective of this project was to:

- Identify fraud trends and patterns
- Analyze customer demographic behavior
- Understand transaction activities
- Detect high-risk categories
- Track fraud activities across locations and time
