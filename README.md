# Fraud Detection Dashboard 🏦

## Project Overview
This project focuses on analyzing transaction data to identify fraudulent activities and understand patterns associated with fraud.

The dashboard provides insights into fraud trends, transaction behavior, and risk indicators to support better monitoring and decision-making.

## Objectives
- Identify fraudulent transaction patterns
- Analyze high-risk transactions
- Monitor fraud trends over time
- Compare fraudulent and non-fraudulent activities
- Support data-driven fraud prevention strategies

## Key Insights
- Identified patterns in fraudulent transactions
- Highlighted high-risk transaction categories
- Detected unusual transaction behavior
- Provided insights for improving fraud monitoring
- Supported better risk assessment and decision-making

## Tools & Technologies
- Power BI
- DAX
- Data Visualization
- KPI Analysis
- Data Modeling

## Dashboard Features
- Interactive dashboard
- KPI cards and fraud metrics
- Transaction trend analysis
- Fraud category analysis
- Visual comparison of fraud patterns
